# kraken-deno

GitKraken cli for Windows and Mac

## Install

```sh
deno install --allow-run --allow-env kraken https://denopkg.com/yuta0801/kraken-deno/index.ts
```

## Usage

```sh
# Just start GitKraken
kraken
# Open GitKraken with repo
kraken path/to/repo
# or current directory repo
kraken .
```
